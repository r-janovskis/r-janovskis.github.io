SIM Card Checker is an application that I created to automate SIM Card file system checking against specification. As a result we were able to reliably check if SIM cards we received from vendors matched our expectations.

Short description how SIM Card Checker worked:

  • Select specification file, where content of each file was defined

  • SIM card gets activated and authenticated with necesarry keys (PIN, PUK codes)

  • We ittereate through each file and compare if it's content mathces content in specification

  • As output HTML file was generated with results.
